# Social-Media-Platform-Lite
A lightweight social media platform built with PHP, MySQL, HTML, CSS, and JavaScript. Features user authentication, post creation, like and unlike functionality, and responsive design. Perfect demonstration of my full stack web development skills.

## Features
### User Authentication
- User Registration and Login system
- Session management for login
- Logout functionality

### Post Management
- Create Posts
- View Posts
- Delete Posts

### Social Interactions
- Like system
- Real time updates
- Interaction tracking
-
